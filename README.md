# tConfig Wrapper
tConfig Wrapper is a mod that allows for tConfig mods to be run within tModLoader.

### Progress
```diff
# Done:
+ Prefixes
+ Assembly Loading

# Doing:
! NPCs
! Items
! Tiles
! Projectiles
! Walls
! IL Patching
! Script Loading

# Haven't started:
- Buff loading
- Sound loading
- Gore loading
- Global Classes
- Generic Classes
```
